The Vault

Secure Currency Storage for Sun Haven

The Vault is a powerful mod that automatically stores your seasonal tokens, community tokens, and keys in a secure digital vault. No more cluttered inventories - your currencies are safely stored and automatically spent when needed.

Key Features

📥

Auto-Deposit

Tokens and keys automatically go to your vault when picked up, keeping your inventory clean.

🛒

Seamless Shopping

Shops and vendors automatically recognize vault currencies. Buy items without manual withdrawals.

🗝

Door Integration

Keys in your vault work with locked doors and chests - no need to carry them in inventory.

👤

Per-Character

Each character has their own vault. Your currencies are kept separate between saves.

🔐

Encrypted Storage

Vault data is encrypted to prevent tampering. Your currencies are safe and secure.

📊

Easy Withdrawals

Need physical items? Quickly withdraw with -1, -5, or -10 buttons directly to inventory.

How to Use

1

Open the Vault

Press Ctrl + V to open the vault interface at any time.

2

View Your Currencies

Switch between tabs (Seasonal, Community, Keys) to see your stored amounts.

3

Automatic Deposits

Simply play the game! Tokens and keys are automatically stored when picked up.

4

Withdraw When Needed

Use the -1, -5, or -10 buttons to withdraw items to your inventory.

Installation

📦

1. Install BepInEx

Download and install BepInEx 5.x for Unity games if you haven't already.

📥

2. Download The Vault

Get TheVault.dll from Nexus Mods.

📁

3. Install the Mod

Place TheVault.dll in your BepInEx/plugins/TheVault folder.

🎮

4. Play!

Launch Sun Haven and press Ctrl+V to access your vault.

Configuration

Setting Default Description
ToggleKey V Key to open/close the vault UI
RequireCtrlModifier true Require Ctrl key when pressing toggle key
EnableAutoSave true Automatically save vault data periodically
AutoSaveInterval 300 Auto-save interval in seconds (5 minutes)

Configuration file located at: BepInEx/config/com.azraelgodking.thevault.cfg
