# The Vault

A Sun Haven mod that adds a spell-driven vault currency system.

## Features

- Adds a vault currency spell for tracking stored funds.
- Integrates with in-game UI to display vault balances.
- Stores vault data alongside character saves for consistency.

## Installation

1. Install [BepInEx](https://github.com/BepInEx/BepInEx) for Sun Haven.
2. Build the mod or download the release.
3. Copy `TheVault.dll` to `Sun Haven/BepInEx/plugins/TheVault/`.
4. Launch the game.

## Configuration

After first launch, edit the config file at:
`Sun Haven/BepInEx/config/com.azraelgodking.thevault.cfg`

You can:
- Toggle the vault spell on or off.
- Adjust default vault currency values.
- Enable or disable UI hints.
