# Sun Haven Mods

A collection of Sun Haven mods maintained in this repository, including HavensBirthright and TheVault.

## Mods

- **HavensBirthright**: Adds unique racial bonuses and traits for each playable race.
- **TheVault**: Adds a spell-driven vault currency system.

## Installation

1. Install [BepInEx](https://github.com/BepInEx/BepInEx) for Sun Haven by following its installation guide.
2. Build the mod you want or download its release.
3. Copy the mod DLL into the appropriate plugin folder:
   - `Sun Haven/BepInEx/plugins/HavensBirthright/`
   - `Sun Haven/BepInEx/plugins/TheVault/`
4. Launch the game.

## Repository Layout

- `HavensBirthright/` contains the HavensBirthright mod.
- `TheVault/` contains the TheVault mod.
- `docs/` contains shared documentation.
